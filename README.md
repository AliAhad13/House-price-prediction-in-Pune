
# 🏠 Pune House Price Prediction

An end-to-end machine learning project that predicts house prices in Pune using real estate data.

## 📊 Project Overview
This project uses features like location, BHK, area, and amenities to predict property prices.

## ⚙️ Features
- Data cleaning and preprocessing
- Feature engineering
- Gradient Boosting model
- Streamlit web app

